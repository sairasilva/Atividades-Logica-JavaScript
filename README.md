# 🚀 Atividades de Lógica JavaScript - PROA

Projeto desenvolvido durante a formação no **PROA**, com foco no desenvolvimento da lógica de programação utilizando **JavaScript**, **HTML** e **CSS**.

A aplicação reúne uma ampla coleção de exercícios interativos organizados por módulos, permitindo praticar desde conceitos básicos até estruturas mais avançadas da linguagem, tudo diretamente no navegador.

---

# 📖 Sobre o projeto

Este projeto foi criado com o objetivo de consolidar os conhecimentos adquiridos durante a formação, oferecendo uma plataforma prática para resolução de exercícios de lógica de programação.

Cada atividade pode ser executada diretamente pela interface, utilizando recursos nativos do JavaScript, como `prompt`, `alert` e `console`, proporcionando uma experiência simples e eficiente para o aprendizado.

Além dos exercícios, a aplicação conta com uma interface moderna, responsiva e intuitiva, incluindo alternância entre tema claro e escuro para melhorar a experiência do usuário.

---

# ✨ Funcionalidades

* ✅ Interface moderna, intuitiva e responsiva

# 🧠 Conteúdos abordados

Os exercícios contemplam diversos conceitos fundamentais da programação:

* Variáveis
* Entrada e saída de dados
* Operadores aritméticos
* Estruturas condicionais
* Operadores lógicos
* Estruturas de repetição (`while` e `for`)
* Vetores (listas)
* Manipulação de arquivos
* Resolução de problemas utilizando lógica de programação
